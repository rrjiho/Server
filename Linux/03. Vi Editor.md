## Vi Editor (Visual Editor)
Linux 및 Unix 시스템에서 사용하는 기본 텍스트 편집기로 강력한 기능을 제공하는 터미널 기반의 편집기  

### Vi 주요 특징
- 모드 기반 편집기: Vi는 세 가지 주요 모드를 제공한다.
  - 명령 모드 (Command Mode): 기본 모드로 명령을 입력하여 텍스트를 편집
  - 입력 모드 (Insert Mode): 실제로 텍스트를 입력할 수 있는 모드
  - 명령행 모드 (Command-Line Mode): 저장, 종료, 검색 등을 수행할 때 사용
- 가볍고 빠름: 텍스트 편집에 특화된 가벼운 프로그램으로 리소스를 적게 사용한다.
- 광범위한 커맨드 지원: 다양한 편집 명령을 사용해 복잡한 텍스트 편집을 효율적으로 수행할 수 있다.

### 기본 명령어
#### Vi 시작 및 종료
- `vi filename`: 파일 열기 (존재하지 않으면 새 파일 생성)
- `:w`: 파일 저장
- `:q`: Vi 종료
- `:wq`: 저장 후 종료
- `:q!`: 변경 사항 무시하고 강제 종료

#### 모드 전환
- `Esc`: 명령 모드로 전환
- `i`: 입력 모드 진입 (커서 앞에 입력)
- `a`: 입력 모드 진입 (커서 뒤에 입력)
- `o`: 새로운 줄에 입력 모드 진입

#### 기본 편집 명령
- `x`: 커서 위치의 문자 삭제
- `dd`: 현재 줄 삭제
- `yy`: 현재 줄 복사
- `p`: 붙여넣기
- `u`: 마지막 명령 실행 취소

#### 탐색 명령
- `h`: 왼쪽으로 한 칸 이동
- `l`: 오른쪽으로 한 칸 이동
- `j`: 아래로 한 줄 이동
- `k`: 위로 한 줄 이동
- `:n`: n번째 줄로 이동 (:10은 10번째 줄로 이동)
- `/pattern`: pattern 검색
- `n`: 검색된 패턴의 다음 항목으로 이동  
초기에는 이렇게 상하좌우 움직였으나 최근에는 통상적인 키로도 매핑이 되었음  
방향키, [home][end] 행의 앞/끝 이동, [page up][page down] 화면 위/아래

#### 텍스트 조작
- `cw`: 단어 바꾸기
- `dw`: 단어 삭제
- `r`: 현재 문자 대체 (커서의 글자 수정)
- `s`: 현재 문자 삭제 (ESC 입력할 때까지 입력)
- `R`: 입력 모드에서 덮어쓰기
